ErEnUa by TheEyes

An IRC bot with power and modules.

ErEnUa is a bot that is easy to code with, secure and doesn't crash all the time.

Dependencies

Python 2.7 pymongo

Install

1. Clone this repository 2. Copy ExampleBot.py to BotName.py (E.G ErEnUa.py) 3. Edit BotName.py as you want. 4. a. Linux: chmod +x startbot ; ./startbot BotName.py or Linux: chmod +x startbot ; cp startbot /usr/bin/erenua ; erenua botname.py b Windows: python botname.py
